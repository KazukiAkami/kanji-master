# 漢字マスター

小学生向けの漢字書き取り練習アプリです。複数の問題集（定例試験など）から選んで練習できます。iPhoneのSafariや、ファイルを直接開く（オフライン）形でも動作します。

## 特徴

- 📱 iPhoneのSafariで使用可能
- 📚 複数の問題集をプルダウンで切り替え
- ✏️ 漢字書き取り問題
- ✅ 問題を見て即座に次へ進める
- 📊 正解数の記録（問題集ごと）
- 🔄 間違えた問題の復習機能（問題集ごと）
- ⬅️ 前の問題に戻る機能
- 💾 ビルド不要・オフライン対応

## 使い方

1. ホーム画面のプルダウンで問題集をえらぶ
2. 「学習をはじめる」をタップ
3. 問題を読んで漢字を考える
4. 「答えを見る」で答えを確認、または「次の問題に進む」で正解としてスキップ
5. 答えを見た場合は「✓ 正解！」or「✗ まちがえた」を選択
6. 全問解答すると結果が表示されます

## 開き方

- **ローカル（オフライン）**: `index.html` をブラウザにドラッグ＆ドロップ、またはダブルクリックで開く。サーバー・ネット接続は不要です。
- **公開（GitHub Pages）**: 後述の手順で公開したURLを開く。

## 問題集の追加・編集

問題データは **`kanji-data.js`** にまとまっています。問題集を追加するときは、このファイルだけを編集します（`index.html` は触りません）。

1. `kanji-data.js` を開き、`KANJI_DATA` の中（囲みのバッククオート `` ` `` の間）に、次の形式で追記します。**クオートは一切不要です。**

   ```
   ## 定例試験（2026/07/01）
   お店をイテンする。,移転
   習い事をヤめる。,辞める
   ```

2. `## タイトル` の行で新しい問題集が始まります。このタイトルがプルダウンに表示され、進捗の保存キーにもなります。
3. それ以降の行は `問題,答え`（**最初のカンマ**で分けます）。空行は無視されるので、問題集の間を空けて見やすくできます。見出し行（`question,answer` など）は不要です。
4. 追記して保存すると、ホーム画面のプルダウンに自動で増えます。

### 問題の書き方（表記ルール）

問題集は次の形式で統一しています。

- **書き取り**: 漢字に直す部分を問題文の中で **カタカナ** で書き、答え列にその漢字を書く（送り仮名はひらがな）。
  例: `習い事をヤめる。,辞める` ／ `山の風がココロヨい気分だった。,快い`
- **読み**: 問題文の頭に `【読み】` を付け、答え列に読みを **カタカナ** で書く（読ませる対象だけカタカナ）。
  例: `【読み】店を構える,店をカマえる`
- **四字熟語**: 問題文の頭に `【四字熟語】` を付け、意味を問題文に、四字熟語を答え列に書く。
  例: `【四字熟語】言葉を口にしなくても相手に気持ちが伝わる事,以心伝心`

### 注意

- 囲みのバッククオート `` ` ``（`KANJI_DATA` の先頭と末尾の1組）は触らないでください。データに **バッククオート `` ` `` と `${`** を含めないでください（文字列が壊れます）。漢字書き取り問題ではまず出てきません。
- 問題文の中の読点は **全角「、」** を使ってください（半角カンマ `,` は問題と答えの区切りに使われます）。
- `## タイトル` は進捗の保存キーになります。一度公開した問題集のタイトルは後から変えないでください（進捗が引き継げなくなります）。

## GitHub Pages で公開する

1. GitHubでリポジトリを作成し、全ファイルをアップロードします。
2. リポジトリの Settings → Pages → Source を **Deploy from a branch** にし、ブランチ `main` / フォルダ `/(root)` を選んで Save。
3. 数分後に `https://<ユーザー名>.github.io/<リポジトリ名>/` で公開されます。
4. iPhoneのSafariでアクセス → 共有ボタン（□↑）→「ホーム画面に追加」でアプリのように使えます。

すべてのパスは相対パスなので、サブフォルダ配信（`/<リポジトリ名>/`）でもそのまま動きます。

## ファイル構成

- `index.html` — 画面とスクリプトの読み込み
- `app.js` — アプリのロジック（クイズ、進捗、問題集の切り替え、データ文字列のパース）
- `kanji-data.js` — 問題集データ（ここを編集して問題集を追加）
- `style.css` — スタイル
- `manifest.json` — PWAマニフェスト
