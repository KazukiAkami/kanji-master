# 漢字マスター

小学5年生向けの漢字学習アプリです。iPhoneのSafariで動作します。

## 特徴

- 📱 iPhoneのSafariで使用可能
- ✏️ 56問の漢字書き取り問題
- ✅ 問題を見て即座に次へ進める
- 📊 正解率の記録
- 🔄 間違えた問題の復習機能
- ⬅️ 前の問題に戻る機能

## 使い方

1. 「学習をはじめる」をタップ
2. 問題を読んで漢字を考える
3. 「答えを見る」で答えを確認、または「次の問題に進む」で正解としてスキップ
4. 答えを見た場合は「✓ 正解！」or「✗ まちがえた」を選択
5. 全56問を解答すると結果が表示されます

## iPhoneのホーム画面に追加

1. Safariでこのページにアクセス
2. 画面下部の共有ボタン（□↑）をタップ
3. 「ホーム画面に追加」を選択
4. 「追加」をタップ

アプリのようにホーム画面から起動できます！
